# switchwon-exercise

### Prerequisites

아래 환경에서 개발되었습니다.

- Node.js v20.19.0
- npm 10.8.2

### 1. Repository Clone

```bash
git clone <repository-url>
cd switchwon-exercise
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Run Development Server

```bash
npm run dev
```

브라우저에서 아래 주소로 접속합니다.

```
http://localhost:3000
```
